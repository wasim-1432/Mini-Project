# Dr. Abul Kalam Memorial Hospital  
### Online OPD Appointment & Token System (2025 Modern Edition)

A beautiful, fast, fully offline-capable OPD appointment booking system specially designed for small to medium hospitals and clinics.

Works instantly – just open `index.html` in any browser!

---

### Features

- Select Doctor (3 doctors pre-added)
- Enter Patient Name, Date & choose Time Slot
- Auto-generated unique Token Number (per doctor + per date)
- One-click PDF Token Slip download (100% working, no logo required)
- View all Today's Appointments in real-time
- Delete any appointment with confirmation
- All data saved automatically in browser (localStorage) – No server needed
- 100% Offline – Works without internet
- Fully Responsive – Looks stunning on Mobile, Tablet & Desktop
- Modern 2025 Design with smooth animations, glassmorphism & gradients

---

### Technology Used

- HTML5
- CSS3 (Flexbox, Grid, Glassmorphism, Animations)
- Vanilla JavaScript (No frameworks)
- jsPDF (for generating PDF token slips)
- localStorage (for data persistence)
- Google Fonts – Inter (clean & professional)

### Easy Customization

| What to Change           | Where to Edit                          |
|--------------------------|----------------------------------------|
| Hospital Name / Address  | `index.html` (header) + `script.js` (PDF section) |
| Logo                     | Replace `hospitallogo.jpg`             |
| Colors & Theme           | Edit `:root` variables in `styles.css` |
| Time Slots               | Edit `timeSlots` array in `script.js`  |
| Doctor List              | Edit `<select>` in `index.html`        |
